# 思考を現実化する手帳

理念 → ビジョン → 目標 → 計画 → 実践 → 振り返り の流れで人生設計・目標達成を記入できる、単一 HTML ファイルのプランナーアプリです。入力内容はブラウザに自動保存され、Google スプレッドシートと双方向で同期できます。

## 使い方

`index.html` をブラウザで開くだけで記入・自動保存・Excel 書き出しが使えます。スプレッドシート連携込みで使うには、下記の手順で Vercel に公開します。

## 収録シート

自分100年史 / 人生でやりたいこと100リスト / 目標の構造化(7分野×理想・現状・GAP) / プライムタイムシート(緊急度×重要度) / 中長期行動計画(10年) / 年間行動計画 / 月間プランナー / ウィークリープランナー / デイリープランナー / 自己宣言文(アファメーション)。月間・週間・日間は日付を切り替えて複数期間を記録できます。

## データの保存

- **自動保存**: 入力はブラウザ(localStorage)に自動保存されます。
- **Excel 書き出し**: 全シートを 1 つの .xlsx としてダウンロードできます。
- **TSV コピー**: 表示中のシートをコピーしてスプレッドシートに貼り付けできます。
- **JSON バックアップ**: 設定タブから書き出し・復元ができます。

## Google スプレッドシート連携(双方向)

「スプレッドシートへ送信」で閲覧用の 10 シートと全データ(隠しシート `_data`)を書き込み、「スプレッドシートから読込」でいつでも復元できます。アプリ起動時にスプレッドシート側が新しければ、確認のうえ自動で読み込みます。別 PC・別ブラウザへのデータ引き継ぎにも使えます。

仕組み: アプリ → Vercel の中継 API(`api/save.js` / `api/load.js`) → スプレッドシート内の Apps Script(GAS)。GAS の URL は Vercel の環境変数に 1 回設定するだけで、ブラウザ側での設定は不要です。Google サービスアカウントや Google Cloud の設定は使いません。

### 初回設定

**ステップ1: スプレッドシートに GAS を設置(約5分)**

1. Google スプレッドシートを新規作成する(名前は例:「思考を現実化する手帳」)
2. メニュー「拡張機能」→「Apps Script」を開く
3. エディタの内容を `gas/Code.gs` の内容に置き換えて保存する
4. 「デプロイ」→「新しいデプロイ」→ 種類「ウェブアプリ」を選び、次のユーザーとして実行:「自分」、アクセスできるユーザー:「全員」でデプロイする
5. 表示された**ウェブアプリの URL** をコピーしておく(次のステップで使います)

**ステップ2: Vercel に公開**

1. このリポジトリを GitHub にプッシュする
2. [Vercel](https://vercel.com/) で「Add New → Project」→ リポジトリを Import する
3. 「Environment Variables」に以下を設定する:

| 変数名 | 値 |
|---|---|
| `GAS_URL` | ステップ1でコピーしたウェブアプリの URL |
| `ACCESS_TOKEN` | 任意の合言葉(推奨。設定すると、同じ値を入力した人だけが読み書きできます) |

4. 「Deploy」を押す。公開 URL を開けば、そのまま同期が使えます
5. `ACCESS_TOKEN` を設定した場合は、アプリの「⚙ 設定・保存」タブの「アクセストークン」に同じ値を入力し、「接続テスト」で確認する

GAS のコードを更新した場合は、Apps Script 側で「デプロイ」→「デプロイを管理」→ 鉛筆マーク →「新しいバージョン」で再デプロイしてください(URL は変わりません)。

### ローカルファイルからの利用

パソコン上の `index.html` を直接開いて使う場合は、「⚙ 設定・保存」タブの「APIのURL」に Vercel の URL(`https://あなたのアプリ名.vercel.app`)を入力してください。

### 注意

- データ本体はスプレッドシートの隠しシート `_data` に保存されます。削除しないでください。
- `ACCESS_TOKEN` を設定しない場合、Vercel の URL を知っている人は誰でもデータを読み書きできます。設定を推奨します。
- GAS のウェブアプリ URL(`GAS_URL`)は Vercel の環境変数の中だけに置かれ、ページには公開されません。

## 技術構成

依存ライブラリなしの単一 HTML ファイル(Excel 書き出しのみ [SheetJS](https://sheetjs.com/) を CDN から使用)。Google 連携は「Vercel Serverless Functions(依存ライブラリなしの中継 API)+ Google Apps Script」で実現しています。
